# Мой день

Простое устанавливаемое веб-приложение для лекарств и встреч. Работает без регистрации, хранит данные на устройстве и открывается офлайн.

**Приложение:** https://antarnyeserdca-dev.github.io/my-day/

## Локальный запуск

```sh
cd /Users/nikitasudian/my-day
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Установка на iPhone

Для установки нужен HTTPS-адрес опубликованного приложения.

1. Открыть адрес в Safari.
2. Нажать «Поделиться».
3. Выбрать «На экран Домой».
4. Запустить приложение с появившейся иконки.

## Напоминания

В разделе «Ещё» можно выгрузить лекарства и встречи в файл `moy-den.ics`. После добавления файла в системный календарь iPhone напоминания сработают и при закрытом приложении.

Уведомления самого веб-приложения используются как дополнительный канал и в текущей автономной версии проверяются, пока приложение запущено. Для фоновых push-уведомлений без календаря потребуется небольшой сервер-планировщик.

## Данные

Данные хранятся в `localStorage`. В разделе «Ещё» доступны экспорт и восстановление резервной копии JSON.
